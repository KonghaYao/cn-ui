# 如何编写一份组件文档

## 标题与信息

1. 一份好的组件文档，它的标题（title）必须包含中文的组件名称和在实际使用情况下的名称，这样便于寻找。

2. 描述信息（description）包含简单的文本说明，用于描述使用场景

3. 附加信息：
   1.  Import: 导入组件的方式
   2.  Source: 源代码位置
   3.  Docs: 文档源码位置
